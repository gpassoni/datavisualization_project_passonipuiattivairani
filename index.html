Human Development Reports

Welcome

With this project we want to show how to the human wealth changed among the years.

To do so we used different data and visualization.

Health Development Index (HDI)

Through a Choropleth map we create a render of the various HDI per country, the color palette rappresent the classification of the nation. Unfortunately the map use the iso format (acronym of the country) you can check the name of the state inserting the iso in the search bar below. The use of this visualization is to show how the different countries are ranked in order of Human Developmenrt Index

Material Foot Print (MF).

We create a map to visualize the footprint changes among the years from 1990 to 2021.

This Choroplet is aiming to show the change of MF throughout the 31 years of data we have.

In the following dorpdown menu we can select different variables for different parts of the world (In some cases continents in in oder big portion of them, e.g. Arab States)
