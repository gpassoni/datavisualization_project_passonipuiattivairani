Welcome

Welcome to our data visualization website, showcasing the progress and developments in health over the years across countries.

You will find interactive graphs and charts that illustrate the changes and trends in various indicators such as Human Development Index (HDI), Material Foot Print(MFT) and and much more information. Explore how various factors such as advances in medicine, pollution, and salary have affected global human development. Get a comprehensive understanding of the challenges and achievements and how they have impacted people's lives.

The target of this website is to inform everyone about the changes appens to the world wealth in the last 30 years. This could be helpful to the companies for instance to understand how the world is changing forecasting the feature cost of moving to a specific country.

Visualization Protocol

Human Development Index (HDI)

Through a Choropleth map we create a render of the various HDI per country, the color palette rappresent the classification of the nation. Unfortunately the map use the iso format (acronym of the country) you can check the name of the state inserting the iso in the search bar below. The use of this visualization is to show how the different countries are ranked in order of Human Developmenrt Index

This plot aims at showing the difference in the Human Development Index across the globe (thus having globe shape). The darker the colour, the higher the Development Index. Since it is an interactive plot, the user (you) can easily navigate it by clicking on it and dragging in a specific direction to make in turn. It is also possible to zoom in. If the cursor hovers a country the ISO code and the HDI amount are displayed. Underneath there is an input text box in which u can put the ISO code u read to get the country's name (in case you don't know what country it is).

Visualization Protocol

Material Foot Print (MF)

We create a map to visualize the footprint changes among the years from 1990 to 2021. This Choroplet is aiming to show the change of MF throughout the 31 years of data we have.

This plot's purpose is to let the user understand the change in the material footprint of each country in the last 30 years. The material Footprint is the amount of a country's resources that are utilized by that same country, hence the amount of materials and resources a country uses for itself. By clicking the play button the plot will change colour, becoming darker if the amount of MF increases and lighter if it decreases.

In the following dorpdown menu is possible to select different variables and return the data divided by the main region of the world.

This section is containing more plots because we wanted to show the difference in the other features among different continents (or parts of the world). By selecting a feature (with the dropdown menu) a specific graph will show up representing the feature selected and its value and change over the years. Each colour represents a group of countries, if you double click on a specific field name in the legend on right side it will show only that selected.
